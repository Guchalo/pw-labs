O CSS (Cascading Style Sheets) estiliza o conteúdo da página web

CSS Seletores
Universal ✔ Aplica o estilo a todos os elementos da página
Tipo ✔ Usa o nome do marcador a estilizar
Classe ✔ Associa um nome a um estilo
Id ❌ Permite identificar univocamente um elemento do HTML (muito útil no JavaScript)
Atributo ✔ Seletores que se aplicam a atributos de elementos
Pseudo-classe ❌ Seletor associado ao estado de um elemento
Pseudo-elemento ✔ Seletor associado ao estado de um elemento
CSS combinadores de seletores
Seletores descendentes ✔ Todos os p que estão “dentro” de um div
Seletores filhos ✔ Todos os p que estão “logo dentro” de div
Seletores adjacentes ✔ Todos os p que estão colocados depois de um div
Seletor imediatamente adjacente ❌ Todos os p que estão colocados “imediatamente” depois de um div
Agrupar os seletores ✔ Separa-os por virgulas
